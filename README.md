# react-native-cloudpayments

CloudPayments SDK fro React Native

## Installation

```sh
npm install react-native-cloudpayments
```

## Usage

```js
import Cloudpayments from "react-native-cloudpayments";

// ...

const result = await Cloudpayments.multiply(3, 7);
```
